Project Description

Search Advanced Technology (SAT) is a leading provider of technology solutions in Sri Lanka. I was tasked with developing an online store for SAT using WordPress. The website features a clean, modern design that showcases SAT's products and services. I implemented WooCommerce to handle product listings, shopping cart functionality, and payment processing. The website is fully responsive and optimized for search engines, ensuring a seamless user experience across all devices.

TECH STACK:

HTML5, CSS3, JavaScript, JQuery, PHP, WordPress, WooCommerce, nginx
